# Task1
Customer Personality Analysis – Data Cleaning and Preprocessing
Data Cleaning Steps Performed:

1. Checked for missing values.
2. Filled missing Income values using median.
3. Removed duplicate records.
4. Standardized Education and Marital_Status values.
5. Converted Dt_Customer column to datetime format.
6. Renamed all columns to lowercase.
7. Created Age column from Year_Birth.
8. Removed unrealistic age values.
9. Saved cleaned dataset as customer_personality_cleaned.csv.

Outcome:
Prepared a clean and analysis-ready dataset.
